# ☕ Coffee Shop Website

A modern and responsive coffee shop website built using **HTML**, **CSS**, and **JavaScript**.  
Designed to provide a clean user experience with smooth animations and a stylish coffee-shop inspired interface.

## 🌐 Live Demo

🔗 https://emadsiddig.github.io/cofe-shope/

## 📌 Features

- Responsive design for mobile, tablet, and desktop
- Modern UI/UX design
- Interactive navigation menu
- Smooth scrolling
- Coffee products showcase
- About section
- Contact section
- Hover animations and effects

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```bash
cofe-shope/
│
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/EmadSiddig/cofe-shope.git
```

Open the project folder:

```bash
cd cofe-shope
```

Run the project by opening `index.html` in your browser.
